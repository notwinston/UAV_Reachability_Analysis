# Gazebo Scenario Recordings

These scenarios use larger east-west separation:
- Attacker starts on the west side of the map
- Defender starts much closer to the goal on the east side
- Main `*.mp4` files are the zoomed versions
- Matching `*_wide.mp4` files keep the full overview-camera render
